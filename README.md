# NestJS Auth Task API

A simple REST API built with **NestJS** that implements **JWT authentication** and **Task CRUD operations**.

This project demonstrates basic backend architecture using NestJS, Passport authentication, and modular structure.

---

## Features

- JWT Authentication
- Passport Local Strategy
- Protected routes with JWT Guard
- Task CRUD API
- DTO validation using class-validator
- Modular NestJS architecture
